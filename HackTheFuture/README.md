
  # Hospital Discharge Summary App

  This is a code bundle for Hospital Discharge Summary App. The original project is available at https://www.figma.com/design/7ufraDNrsM7c60Q5rRLFKp/Hospital-Discharge-Summary-App.

## Setup

Install dependencies:

```bash
npm install
```

## Running the app

Start the API/server:

```bash
npm run server
```

In another terminal, start the frontend dev server:

```bash
npm run dev
```

## Notes

- If you need environment variables, copy `.env.example` to `.env` and fill in values.
